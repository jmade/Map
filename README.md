
## Overview 

A map generator using [Voronoi Tessellation](https://en.wikipedia.org/wiki/Centroidal_Voronoi_tessellation).

![Alt Text](https://github.com/jmade/jmade.github.io/blob/master/map.gif?raw=true)

## Grand Central Dispatch 

This was my first time using `GrandCentralDispatch` and asynchronous drawing with the `CoreGraphics` framework. 
I also wrote a wrote a very simple random line function to make the lines jagged looking.
